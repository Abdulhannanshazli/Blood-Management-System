# Blood-Management-System
A C++ Blood Management System project is designed to manage blood donor and recipient data. It allows users to insert, store, and retrieve information about donors, recipients, and blood types. 
This C++ Blood Management System project is designed using Object-Oriented Programming (OOP) principles to manage blood donors and recipients efficiently. The system allows users to:

Add Donors and Recipients: Users can input donor and recipient details, including their name, blood type, and contact information.

View Records: Users can display all stored donor and recipient data, ensuring easy access to the relevant information.

Find Compatible Donors: The system has a blood type compatibility feature, allowing users to search for compatible donors based on a recipient's required blood type. For instance, universal donors (O-) and recipients (AB+) are handled correctly.

OOP Concepts:
Encapsulation: Data for donors and recipients are stored in private members, accessible only through public methods.
Abstraction: The complexity of matching blood types and managing records is hidden behind simple functions.
Modularity: Each class (Donor, Recipient, BloodManagementSystem) is designed for a specific role, making the system modular and easy to extend or modify.
